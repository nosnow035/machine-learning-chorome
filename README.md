# machine-learning-chorome
大学で機械学習のプロジェクトを行いました。
機械学習を用いた振り仮名chorome拡張機能です。

モデルとデータセットの生成は、Google Colabで行いました。

サーバーもgoogle colabで行い、フロントエンドは途中ですがjsとhtmlで行いました。

データセットは漢字文と振り仮名文を形態素解析のsudachiを解答として学習させた。
モデルはT5を使用
